# 📊 Customer Churn Analysis Dashboard

An interactive **Customer Churn Analysis Dashboard** built in **Power BI** to analyze customer behavior, identify churn patterns, and provide actionable business insights. The dashboard enables stakeholders to monitor customer retention, understand churn drivers, and make data-driven decisions.

---

## 📌 Project Overview

Customer churn is one of the most important metrics for subscription and service-based businesses. This dashboard provides a comprehensive overview of customer demographics, contract types, internet services, payment methods, and churn statistics through interactive visualizations.

## 📌 Insights Generated

- Month-to-Month contracts have the highest churn rate.
- Customers using Fiber Optic internet service show higher churn compared to DSL.
- Male and Female churn distribution is nearly equal.
- Customers with higher monthly charges are more likely to churn.
- Long-term contracts significantly improve customer retention.
